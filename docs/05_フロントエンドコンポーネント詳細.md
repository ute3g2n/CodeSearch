## 2. エクスプローラー

### 2.1 ExplorerPanel

**ファイル:** `src/components/explorer/ExplorerPanel.tsx`

```typescript
// ワークスペース未選択時: 「フォルダーを開く」ボタンを表示
// ワークスペース選択済: ファイルツリーを表示
//
// ツリーの初期読み込み: depth=1 でルート直下を取得
// フォルダ展開時: get_file_tree(path, depth=1) で子ノードを遅延取得

interface ExplorerPanelInternalState {
  /** 展開中のディレクトリパスのSet */
  expandedDirs: Set<string>;
  /** ツリーデータ（ルートレベル） */
  tree: FileNode[];
  /** 読み込み中 */
  isLoading: boolean;
}

// ── フォルダのドラッグ&ドロップ ──
// ワークスペース未選択時にエクスプローラーパネル全体が drop zone となる
// onDragOver: preventDefault() でドロップ可能を示す
// onDrop: event.dataTransfer.files[0] のパスを取得し open_workspace() を呼び出す
// ドロップ中はパネルをハイライト表示（点線ボーダー + 薄いオーバーレイ）
```

// - トークナイズは WebWorker で非同期実行
// - 可視範囲 ± 50行 のみトークナイズ
// - トークナイズ結果はLRU(20ファイル分)キャッシュ
// - ハイライトワードマッチは可視範囲の行のみ走査
