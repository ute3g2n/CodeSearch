## 命名規則

| 対象 | 規則 | 例 |
|------|------|-----|
| Rust struct/enum | PascalCase | `SearchResult`, `AppError` |
| Rust function | snake_case | `build_full_index`, `search_fulltext` |
| TypeScript interface | PascalCase | `SearchResult`, `FileNode` |
| TypeScript function | camelCase | `openWorkspace`, `searchFulltext` |
| Zustand store | camelCase + Store | `useSearchStore`, `useEditorStore` |
| React component | PascalCase | `CodeView`, `SearchSection` |
| IPC command | snake_case | `search_fulltext`, `open_workspace` |
| IPC event | namespace://name | `index://progress`, `fs://changed` |

## 依存方向

```
Presentation (React Components)
    ↓
Store (Zustand)
    ↓
IPC Bridge (src/ipc/)
    ↓
Command Router (commands/)
    ↓
Services (services/)
    ↓
Infrastructure (indexer/, storage/, watcher/, encoding/)
```

上位レイヤーは下位レイヤーに依存するが、逆方向の依存は許可しない。
